This text will heartbeat!

  • animation: heartbeat 1s infinite; - This is a shorthand property for setting all the individual animation properties at once (name, duration, timing function, delay, iteration count, direction, fill mode, and play state).
  • @keyframes heartbeat; - This is where we define what the animation should do at certain times. In this case, it scales the size of the text up and down, causing a heartbeat effect.
