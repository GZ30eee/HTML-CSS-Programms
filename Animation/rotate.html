This text will rotate!

  • animation: rotation 2s infinite linear; - This is a shorthand property for setting all the individual animation properties at once (name, duration, timing function, delay, iteration count, direction, fill mode, and play state).
  • @keyframes rotation; - This is where we define what the animation should do at certain times. In this case, it rotates the text from 0 degrees to 359 degrees, causing a rotation effect.
