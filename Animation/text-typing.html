This text will type itself!

  • animation: typing 2s steps(40, end), blink-caret .75s step-end infinite; - This is a shorthand property for setting all the individual animation properties at once (name, duration, timing function, delay, iteration count, direction, fill mode, and play state). Here, we have two animations: one for the typing effect and one for the blinking caret effect.
  • @keyframes typing, @keyframes blink-caret; - These are where we define what the animations should do at certain times. The typing animation increases the width of the text box from 0 to 100%, giving the illusion of typing. The blink-caret animation changes the color of the border, giving the illusion of a blinking caret.
