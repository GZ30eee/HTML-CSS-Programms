This text will blink!

  • animation: blink 1s infinite; - This is a shorthand property for setting all the individual animation properties at once (name, duration, timing function, delay, iteration count, direction, fill mode, and play state).
  • @keyframes blink; - This is where we define what the animation should do at certain times. In this case, it changes the opacity of the text from 1 to 0 and back to 1, causing a blinking effect.
