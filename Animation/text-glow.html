This text will glow!

  • animation: glow 1s infinite alternate; - This is a shorthand property for setting all the individual animation properties at once (name, duration, timing function, delay, iteration count, direction, fill mode, and play state).
  • @keyframes glow; - This is where we define what the animation should do at certain times. In this case, it changes the glow color of the text from green to red, causing a glowing effect.
