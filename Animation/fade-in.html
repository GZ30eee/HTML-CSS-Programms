This text will fade in!

  • animation: fadeIn ease 3s; - This is a shorthand property for setting all the individual animation properties at once (name, duration, timing function, delay, iteration count, direction, fill mode, and play state).
  • @keyframes fadeIn; - This is where we define what the animation should do at certain times. In this case, it changes the opacity of the text from 0 to 1, causing a fade-in effect.
