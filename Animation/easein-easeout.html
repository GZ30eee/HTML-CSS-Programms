Animation Example

@keyframes colorchange defines an animation that changes the background color of the div from red to yellow. @keyframes movediv defines an animation that moves the div from left to right, right to bottom, bottom to left, and left to top. animation: colorchange 2s infinite, movediv 8s infinite; applies both animations to the div, with a delay of 2 seconds. The infinite keyword makes the animation run an infinite number of times. animation-direction: reverse; makes the animation run in reverse order. animation-timing-function: ease-in-out; changes the speed of the animation: it starts slow, speeds up in the middle, and then slows down at the end.
