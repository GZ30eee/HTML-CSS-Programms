The Document Heading

HTML Elements

<title> Element

The <title> tag defines the title of the document. It is required in all HTML documents.


<title>The Document Heading</title>

<script> Element

The <script> tag is used to embed or reference JavaScript code.


<script>
    document.write("Hello, World!");
</script>

<noscript> Element

The <noscript> tag defines an alternate content for users that have disabled scripts in their browser or have a browser that doesn't support script.


<noscript>Your browser does not support JavaScript!</noscript>

<link> Element

The <link> tag defines a link between a document and an external resource. It is most commonly used to link to stylesheets.


<link rel="stylesheet" type="text/css" href="mystyle.css">

<style> Element

The <style> tag is used to define style information for an HTML document.


<style>
    body {background-color: powderblue;}
</style>

<base> Element

The <base> tag specifies the base URL and/or target for all relative URLs in a document.


<base href="https://www.example.com/" target="_blank">

<meta> Element

The <meta> tag defines metadata about an HTML document. Metadata is data (information) about data.


<meta charset="UTF-8">
<meta name="description" content="This is a webpage that describes HTML elements.">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="John Doe">
