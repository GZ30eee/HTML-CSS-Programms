Understanding Framesets in HTML

The <frameset> tag was used in HTML 4 to define a frameset, which is a group of frames. Each frame could hold a separate document. The <frameset> tag could specify the number of rows and columns in the frameset, and how much space they would occupy in a frame. However, it's important to note that the <frameset> tag is not supported in HTML5.

Here's an example of how the <frameset> tag was used:

<!DOCTYPE html>
<html>
<head>
    <title>Frameset Example</title>
</head>
<frameset cols="30%, 40%, 30%">
    <frame src="file1.html">
    <frame src="file2.html">
    <frame src="file3.html">
</frameset>
</html>

In this example, the browser window is divided into three vertical frames of widths 30%, 40%, and 30% respectively. Each frame loads a different HTML document (file1.html, file2.html, file3.html). Please replace these with the actual paths to your HTML files.

Remember, since the <frameset> tag is not supported in HTML5, it's recommended to use the <iframe> tag for embedding documents.
